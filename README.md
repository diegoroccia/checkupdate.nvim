# CheckUpdate.nvim

## Install

* install with [https://github.com/folke/lazy.nvim](lazy.nvim)

```
    {
        "diegoroccia/checkupdate.nvim",
        lazy = false,
        dev = true,
        config = true,
        init = function()
            require "checkupdate".init()
        end
    }
```
